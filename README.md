# 임호영 | Backend & Infra Developer

안녕하세요. 문제를 구조화하고 안정적인 서비스로 개선하는 개발자 임호영입니다.

저는 백엔드 개발을 중심으로 AI 서비스 연동, 인프라 구성, 모니터링까지 경험해왔습니다.  
단순히 기능을 구현하는 것보다, 실제 운영 중 발생할 수 있는 지연, 실패, 데이터 불일치 문제를 로그와 지표로 분석하고 개선하는 데 관심이 많습니다.

최근에는 LLM Agent, MCP, FastAPI, Spring Boot를 활용한 AI 주문 키오스크 프로젝트에서 인프라, 모니터링, 하드웨어 연동을 담당했습니다.
