## About Me

저는 백엔드 개발을 중심으로 **AI 서비스 연동, 인프라 구성, 모니터링, 운영 안정화**에 관심이 있는 개발자입니다.

단순히 기능을 구현하는 것보다, 실제 서비스에서 발생할 수 있는 **지연, 실패, 데이터 불일치, 연동 오류**를 로그와 지표로 분석하고 개선하는 과정에 강점이 있습니다.

최근에는 **LLM Agent, MCP, FastAPI, Spring Boot**를 활용한 AI 주문 키오스크 프로젝트에서 인프라, 모니터링, 하드웨어 연동을 담당했습니다.
